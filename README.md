# Task Manager CLI
C++ task manager (CLI) almost like a todo app

just do
```make run```

to run, do ``./task <command> <name for add and type an id for everything>``

## Commands:
`./task add <taskName>` <br><br>
`./task remove/delete <id#>` <br><br>
`./task mark <id#>` <br><br>
`./task unmark <id#>` <br><br>
`./task list <id#>` <br><br>
